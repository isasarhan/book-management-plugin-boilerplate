=== Plugin Name ===
Book management plugin

this is a plugin developed in boilerplate framework that manages books and book shelfs

== Description ==

the plugin adds sample data to database upon installation.
It utilizes CRUD operations 
Imports bootstrap for utilizes, datatable to show table of books, validate for field validations

Upon adding a book we can choose which shelf to assign for book from the added shelfs to the database
== Installation ==

This section describes how to install the plugin and get it working.


e.g.

1. Upload `book-management.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
